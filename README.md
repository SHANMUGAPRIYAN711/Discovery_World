# 🌍 Discovery World Clone

A fully responsive **Discovery World-inspired website** built using **HTML5, CSS3, and JavaScript**. This project recreates the look and feel of a modern science museum website with a responsive navigation bar, hero slider, feature cards, events section, newsletter, and footer.

> **Note:** This is an educational frontend project created for learning responsive web development. It is not affiliated with or endorsed by Discovery World.

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/DiscoveryWorldClone.git
```

### Navigate to the Project

```bash
cd DiscoveryWorldClone
```

### Open the Project

Simply open the `index.html` file in your browser.

Or use VS Code Live Server.
